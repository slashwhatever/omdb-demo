# OMDB Demo App

### Installation

Clone to a new folder and run:

```
yarn install && yarn run dev

```

Libraries/frameworks used:

React
https://github.com/facebook/react

Axios
https://github.com/axios/axios

Redux Rematch
https://github.com/rematch/rematch

Heroicons
https://heroicons.com/

TailwindCSS
https://tailwindcss.com

Vite
https://vitejs.dev/
